### 2026-06-29 — Spec #013 In-App File Viewers ✅ MERGED (PR #13)

- **In-app viewers shipped** (last v1.0 feature spec before Polish): tapping a **received** known-type file from the History detail list, the Receive-complete list, or the Home recent / See-all media grids now opens it **inside Safe Send** instead of bouncing to the OS share/open sheet.
